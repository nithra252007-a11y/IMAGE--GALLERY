# IMAGE-GALLERY
The Image Gallery project is a responsive and interactive web-based gallery developed using HTML, CSS . It displays images in a well-structured layout that adjusts smoothly across different screen sizes such as desktops, tablets, and mobile devices. CSS is used to add attractive hover effects and smooth transitions.

This is a simple project image gallery created using HTML and CSS.  
It is useful for displaying project images, photos in an organized way.

- Simple and clean design  
- Responsive layout  
- Made only with HTML and CSS  
- Easy to understand and edit
